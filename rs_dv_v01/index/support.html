결제/정산
회원가입은 어떻게 하나요? ▾
상단의 "회원가입" 버튼을 클릭한 후, 3단계(기본 정보 → 계정 유형 → 약관 동의)를 완료하시면 됩니다. 이메일 인증을 통해 가입이 최종 확정됩니다.
비밀번호를 잊어버렸어요. ▾
로그인 페이지의 "비밀번호 찾기"를 클릭하고, 가입 시 사용한 이메일 주소를 입력하시면 비밀번호 재설정 링크를 보내드립니다.
에이전시로 등록하려면 어떤 서류가 필요한가요? ▾
사업자등록증(필수), 회사 소개서(선택), 회사 로고(선택)를 업로드해주세요. 내부 심사 후 최대 3영업일 내에 승인/반려 결과를 알려드립니다.
환자 송출 시 수수료는 어떻게 정산되나요? ▾
예약이 확정되고 진료가 완료되면, 매월 정산일에 합산 정산됩니다. 에이전시 대시보드에서 내역을 실시간으로 확인할 수 있습니다.
병원 예약은 어떻게 하나요? ▾
병원 검색 → 원하는 병원 선택 → 진료과 선택 → 상담 신청. MediW팀이 병원과 연결하여 일정을 조율해 드립니다.
비자·통역·숙소 준비도 도와주나요? ▾
네, 컨시어지 서비스를 통해 비자 신청, 통역사 배정, 숙소 예약, 공항 픽업까지 원스톱으로 지원합니다.
병원으로 플랫폼에 등록하려면? ▾
회원가입 시 "병원"으로 계정 유형을 선택한 후, 병원 등록 페이지에서 세부 정보(주소, 진료과, 인증 정보 등)를 입력해주세요. 심사를 거쳐 활성화됩니다.
진료비는 플랫폼을 통해 결제되나요? ▾
진료비는 병원에 직접 지불하시며, MediW는 상담/매칭 서비스에 대한 수수료만 정산합니다.
