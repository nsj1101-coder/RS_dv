개발사 (MediW)

플랫폼을 만들고, 기술 지원과 마케팅 소재를 제공하는 역할

플랫폼 개발

  • 전체 20개 페이지 설계 · 개발 · 배포 (Vercel)
  • 다국어 UI (EN, KO, JA, ZH, RU) 구현
  • 다크/라이트 테마, 모바일 반응형
  • 환자 케이스 8단계 상태 파이프라인 시스템
  • 어드민 시스템 (회원·케이스·병원·에이전시·승인·설정)

기술 지원

  • 버그 수정, 장애 대응, 서버 모니터링
  • RBG 요청에 따른 기능 추가 · 개선
  • DB 스키마 설계 및 데이터 마이그레이션
  • RBG 운영팀 대상 어드민 사용법 교육
  • 플랫폼 성능 · 보안 유지보수

마케팅 지원

  • 랜딩페이지 콘텐츠 · 디자인 제작
  • SEO 최적화, 검색 광고 세팅
  • Featured Hospitals, 후기, 통계 데이터 관리
  • 글로벌 브랜딩 소재 (배너, 소개 자료, 영상)
  • RBG 현지 마케팅용 번역 원본 · 이미지 제공
2

운영사 (RBG)

플랫폼 위에서 실제 사업을 운영하는 주체 — 병원 · 에이전시 · 환자 모든 접점을 관리 · 감독

병원 관리

  • 한국 병원 영업 · 계약 (아산, 세브란스, 서울대 등)
  • 병원 등록 폼(admin) 직접 수행 및 데이터 입력
  • 진료과 · 교수진 정보 수집 · 정기 업데이트
  • 병원 국제진료센터와 커뮤니케이션 유지
  • 병원 측 승인 프로세스 안내 · 교육

에이전시 관리

  • 에이전시 발굴 · 영업 · 계약
  • 에이전시 플랫폼 가입 유도 및 온보딩
  • 에이전시 프로필 검증 (사업자등록증, 회사소개서)
  • 에이전시 교육 — 접수 방법, 플랫폼 사용법, 서류 기준
  • 에이전시 성과 모니터링 (환자 수, 접수 품질)

케이스 관리 · 승인

  • 어드민에서 전체 환자 케이스 모니터링
  • 접수 → 병원 검토 요청 → 자료 요청 전달 · 독촉
  • 병원 승인 후 환자/에이전시에 최종 확인 요청
  • 분쟁 · 이슈 발생 시 병원-에이전시 간 중재
  • 회원 관리 — 가입 승인, 계정 상태 관리

현지 마케팅 · 영업

  • 개발사 제공 소재를 활용한 현지어 마케팅 집행
  • 의료관광 박람회 · 세미나 참가
  • 신규 국가 확장 시 에이전시 네트워크 개척

정산 · CS

  • 에이전시별 커미션, 병원별 수수료 정산
  • 전체 매출 · 케이스 통계 관리 (어드민 대시보드)
  • 에이전시 · 병원 · 환자 문의 대응 (CS)
3

에이전시

RBG가 모집 · 관리하는 회사들 — 환자를 확보하고 접수 · 동행을 담당

환자 확보

  • 자체 네트워크로 환자 발굴 (병원 연계, 입소문, 마케팅)
  • 환자 상담 — 한국 의료관광 안내, 병원 · 진료과 추천

플랫폼 접수 업무

  • 병원 검색 → 진료과 · 교수진 확인 → 진료과 선택
  • 환자 접수: 기본정보 + 의료정보 + 서류(CT/MRI 등) 업로드
  • 자기 환자 리스트 관리, 케이스 상태 추적
  • 병원 자료 요청 시 환자로부터 서류 수집 · 업로드
  • 병원 승인 후 환자 측 최종 승인 / 철회 결정

컨시어지 서비스 실행

  • 공항 픽업, 차량 서비스, 숙박 수배
  • 통역 서비스 (진료 · 수속 동행)
  • 한국 도착 → 병원 동행 → 입원 케어 → 귀국까지 전 과정
